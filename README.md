📌 About the Project:


Medicare is a full-stack healthcare platform designed to streamline doctor appointments and service bookings.

It includes Admin, Doctor, and Service dashboards with secure payments and authentication — making it a real-world SaaS-level project.



✨ Features Overview



👤 Patient Features:



🔍 Search & Discover
Explore doctors and healthcare services

📅 Appointment Booking
Book appointments quickly with preferred doctors

💳 Secure Payments
Safe and seamless checkout via Stripe

🔐 Authentication
Secure login and signup with Clerk

📱 Responsive Design
Optimized for both desktop and mobile



👨‍⚕️ Doctor Dashboard:


📊 Dashboard Overview
View appointments, earnings, and status summaries

📅 Appointment Management
Accept, reject, or reschedule bookings

🧾 Patient Information
Access patient details and history

💰 Earnings Tracking
Monitor revenue from appointments

⏰ Availability Control
Manage working hours and availability

✏️ Profile Management
Update profile, fees,image,appointment slots and specialization




🧑‍💼 Admin Dashboard:



👨‍⚕️ Doctor Management
Add, remove, and manage doctor listings

🏥 Service Management
Add, edit, and manage healthcare services

📅 Appointment Management
View and manage all appointments (doctor + service)
Approve or cancel bookings

👥 User Management
Manage patient data and system users




💳 Payment System:


✔️ Integrated Stripe payment gateway
✔️ Secure checkout experience
✔️ Real-time payment handling



🔐 Security & Access Control:


🔑 Clerk Authentication system
